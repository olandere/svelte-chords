Chord Charts - beta

Welcome to the Chord Charts site. This site provides string instrument players (guitar, banjo, ukulele, mandolin, etc) with a reference to chord fingerings all over the fretboard. It supports any number of strings and any tuning. This site was an outgrowth of command line tools I developed to help me when I encountered an unfamiliar chord on a jazz chart or needed a fingering for a chord at a particular location the fretboard. It differs from other tools I have seen in that the chord names are parsed so you are not locked into a pre-configured chord list, alternate tunings are supported, it handles chord progressions, shows arpeggio patterns, and can name chords when given a fingering.

Inputs

The Chord field accepts a variety of inputs - Chord names (e.g. Am, GM9, F#m7b5), chord fingerings (x 0 2 2 2 0), pitch class sets {0 1 4 7 9}, comma separated lists of chord names or fingerings, and chord names with any of the commands below prefixed to them.

The Fret span field specifies how wide apart the fingering of the chord is allowed to be. For example, a fretspan of 1 means all the chord tones will be on the same fret (such as Am7 at the fifth fret in standard guitar tuning).

The Tuning field accepts tunings in two formats:
  • List of note names - EADGBE
  • Joni Mitchell notation - E55545
The tuning dropdown list will remember the tunings you enter to make it easy to switch between the tunings you use. You are not restricted to guitar tunings. Banjo, ukulele and other string instruments are supported, just enter the appropriate tuning.

The Shell checkbox will cause the chord generation algorithm to select chords in shell voicing which is the chord with no 5th, the 7 and only the highest extension. As an example an E11 contains the R, 3, 5, dom 7, 9 and 11. The E11 shell contains R 3 7 11. These voicings are commonly used in jazz.

The Jazz voicings only checkbox is probably misnamed, but the idea is to generate chords with no repeated tones. The E major chord most familiar to guitarists (0 2 2 1 0 0) contains 3 E's, 2 B's and 1 G#. Selecting this option will only return 3 note chords with exactly 1 E, G# and B (e.g. x x 2 1 0 x).

Commands

The Chord field entry can be refined with the following commands.
  • root: - Renders the chord with the Root note (R) as the lowest note.
  • drop2: - Renders the chord in Drop 2 voicing.
  • drop24: - Renders the chord in Drop 2 and 4 voicing.
  • shell: - Renders the chord as a shell chord (no 5, and only the highest extension).

Examples

  • Am7 - Renders all possible fingerings of the Am7 chord.
  • Am7@5 - Renders all possible fingerings of the Am7 chord with at least 1 note at the 5th fret.
  • drop2: Am7 - Renders all possible fingerings of the Am7 chord using Drop 2 voicing.
  • Am7, D7, GM7 - Renders the closest fingerings for the three chords making up this progression.
  • root: Am7, drop2: D7, shell: GM9 - Yes, you can get this specific!
  • 0 2 2 1 0 0 - Identifies most likely chord (E) and renders it.
  • 0 2 2 1 0 0, x 0 2 2 1 0 - Identifies and renders this list of chords (E and Am).
  • {0 1 2 7}, A {0 1 2 3 e} - Specify a pitch class set with an optional starting note, which defaults to C if not specified.
(NOTE: The code for the chord progressions currently just uses a distance measure between the chords, so it makes odd choices at times. Ideally it would take voice leading into account too so that they sound 'right'.)

Inputs

The Scale field presents a drop down of common scales (modes of the Major scale and of the Harmonic Minor scale, major and minor pentatonic scales) as well as allowing you to specify any set of scale degrees (for example: C 2 b3 5 b6 gives the Japanese mode starting on C)

The Tuning field accepts tunings in two formats:
  • List of note names - EADGBE
  • Joni Mitchell notation - E55545
The tuning dropdown list will remember the tunings you enter to make it easy to switch between the tunings you use. You are not restricted to guitar tunings. Banjo, ukulele and other string instruments are supported, just enter the appropriate tuning.

Examples

  • C Phrygian - Phrygian mode starting on C
  • D 2 3 #4 #5 #6 - Whole tone scale starting on D
